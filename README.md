A simple practice.  How to write to and read from files in rust.  The filename is constant "./test.dat".

Usage
-----

With [rust](https://github.com/mozilla/rust) master.

    % rustc file_writer.rs
    % ./file_writer
    % rustc file_reader.rs
    % ./file_reader
    90
    22
    %
